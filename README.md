# Notes Saver Chrome Extension

It is a simple Chrome Extension that allows users to save notes using Chrome Storage API.

## Features

* Save notes.
* Automatically load saved notes.
* Simple and lightweight interface

## Technologies Used

* HTML
* CSS
* JavaScript
* Chrome Extension API

## Installation

1. Clone the repository
2. Open chrome://extensions
3. Enable Developer Mode
4. Click Load Unpacked
5. Select the project folder
